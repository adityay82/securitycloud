🔐 Excited to share my internship project — a production-grade cloud security platform built from scratch!

**What it does:**
CloudSec protects cloud user accounts and files using three complementary layers:
1. **3-Level MFA** — Password → Email OTP → ML-triggered risk OTP
2. **Adaptive AES Encryption** — Auto-selects AES-128/192/256 by file size
